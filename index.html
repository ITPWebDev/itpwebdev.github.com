#1 HTML & CSS

HTML

HTML, HyperText Markup Language, is exactly that, a markup language for text on internet. A markup language is a computer language to make text and media readable by a computer.

A word processor like Pages or Word is doing its own markup in the background to give you nice paragraphs and fonts and colors. Copying and pasting from a word processor does not retain this markup and anyway Pages and Word use proprietary markup, so you can't have it.

HTML is made up of text and media enclosed in elements forming a tree structure, which is mimicked in text through the convention of tabbing.

<html>
  <head>
    <title>HELLO WORLD</title>
  </head>
  <body>
    <div>
      <h1>I'm hungry.</h1>
      <p>So give me some food.</p>
    </div>
  </body>
</html>

Elements need to be closed with a </p>. Elements can be nested, enclosed within other elements.

It's really best to show you through an example.
